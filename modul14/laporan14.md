# laporan praktikum modul 14 : WIFI

# Tujuan Praktikum
1. dapat menginvestigasi cara kerja WiFi menggunakan Wireshark

# langkah praktikum
1. Download file zip http://gaia.cs.umass.edu/wireshark-labs/wireshark-traces.zip dan ekstrak file
Wireshark_802_11.pcap. Jejak ini dikumpulkan menggunakan AirPcap dan Wireshark yang berjalan
di komputer di jaringan rumah salah satu penulis, yang terdiri dari titik akses/router gabungan
Linksys 802.11g, dengan dua PC berkabel dan satu PC host nirkabel yang terpasang ke titik

2. Setelah mengunduh jejak, dapat memuatnya ke Wireshark dan melihat jejak
menggunakan menu tarik-turun File, pilih Buka, lalu pilih file jejak Wireshark_802_11.pcap.
Tampilan yang dihasilkan.
